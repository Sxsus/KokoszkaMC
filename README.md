# KokoszkaMC 🐔

Własny, profesjonalny panel hostingowy dla serwerów Minecraft, inspirowany Aternosem. 

## Główne funkcje:
* Pomarańczowy, ciemny interfejs.
* Serwery działające 24/7.
* Wybór silnika (Vanilla, Paper, Forge).
* Wykresy RAM w czasie rzeczywistym.
* Dynamiczne porty i pełne zarządzanie z poziomu strony WWW.

Projekt stworzony w języku Python (Flask) oraz HTML/CSS.
