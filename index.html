🚀 Nielimitowany Przydział Mocy

W KokoszkaMC zasoby są darmowe. Maszyna wirtualna używa jednak fizycznych zasobów Twojej przeglądarki, więc przydzielaj je mądrze!

Przydzielona Pamięć RAM (GB) 4
KRYTYCZNE OSTRZEŻENIE SYSTEMOWE: Wykryto alokację ponad 16GB RAM. Przeglądarka może ulec awarii (Out-Of-Memory), jeśli Twój fizyczny sprzęt nie posiada tyle wolnej pamięci.
Rdzenie Procesora (vCPU) 2
OSTRZEŻENIE O WYDAJNOŚCI: Silnik Java dla Minecrafta wykorzystuje max. 4 rdzenie do głównego wątku. Wyższa wartość sztucznie obciąży scheduler Windowsa/Linuxa.
⚙️ Wybór Oprogramowania

Zmień silnik serwera w dowolnym momencie. System pobierze nowe biblioteki Java .jar z chmury.

🛠️ Ustawienia Mechaniki Gry

Edytuj najważniejsze wartości z pliku server.properties poprzez bezpieczne przełączniki UI.

Tryb Non-Premium (Cracked / Offline-Mode)

Wyłącza weryfikację kont z serwerami Mojang. Wymagane dla graczy bez zakupionej gry.

Walka PvP (Player vs Player)

Umożliwia graczom zadawanie sobie obrażeń za pomocą broni i czarów na mapie.

Biała Lista (Whitelist)

Zablokuj serwer dla nieznajomych. Tylko gracze wpisani na listę będą mogli dołączyć.
